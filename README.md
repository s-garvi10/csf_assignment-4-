# CSF Assignment: Phonebook Application

## Overview
This is a simple command-line Phonebook application for the Computer Science Fundamentals (CSF) assignment.  
It allows users to store, lookup, update, and delete contact information using Python dictionaries and functions.

## Features
- Show all contacts in the phonebook
- Lookup contact by name
- Add new contact/number
- Update existing contact/number
- Delete contact/number
- User-friendly menu interface

## How to Run
1. Make sure Python (3.x) is installed on your system.
2. Clone this repository or download `main.py` to your local machine.
3. Open a terminal and navigate to the project folder.
4. Run the following command:
    ```
    python main.py
    ```
